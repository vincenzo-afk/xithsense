# Domain Glossary

Cricket and fantasy cricket terminology used throughout the XithSense codebase, documentation, and user interface.

---

## Cricket Formats

| Term | Definition |
|------|-----------|
| **T20** | Twenty20 cricket — 20 overs per side, highest-scoring and most popular fantasy format |
| **ODI** | One Day International — 50 overs per side, international matches only |
| **Test** | Five-day format cricket — unlimited overs; oldest format |
| **IT20** | International T20 — men's international T20 (distinct from franchise T20) |
| **IPL** | Indian Premier League — India's flagship T20 franchise tournament |
| **BBL** | Big Bash League — Australia's T20 franchise tournament |
| **PSL** | Pakistan Super League — Pakistan's T20 franchise tournament |
| **CPL** | Caribbean Premier League — West Indies T20 franchise tournament |
| **WPL** | Women's Premier League — India's women's T20 franchise tournament |
| **NTB** | Non-Test-playing nations T20 matches |
| **CCH** | Club/county cricket matches |
| **MDM** | Multi-day matches (non-Test first-class cricket) |

## Match Structure

| Term | Definition |
|------|-----------|
| **Innings** | One team's batting period. T20: 2 innings total; ODI: 2; Test: up to 4 |
| **Over** | 6 legal deliveries bowled by one bowler |
| **Delivery** | A single ball bowled. Also called a "ball" |
| **Legal delivery** | A ball that is not a wide or no ball |
| **Wide** | Delivery too far from the batsman; 1 extra run awarded; not counted as a legal delivery |
| **No ball** | Illegal delivery (front foot, height, etc.); 1 extra run; not counted as a legal delivery |
| **Bye** | Runs scored when the ball passes without touching the bat or batsman |
| **Leg bye** | Runs scored off the body (not the bat) |
| **Extras** | Total of wides + no balls + byes + leg byes on a delivery |
| **Powerplay** | Fielding restriction overs. T20: overs 1–6 (mandatory). ODI: overs 1–10 |
| **Middle overs** | T20: overs 7–15. ODI: 11–40 |
| **Death overs** | Final batting overs. T20: 16–20. ODI: 41–50 |
| **Super Over** | One-over elimination tiebreaker when scores are level at match end |
| **DLS** | Duckworth-Lewis-Stern method — mathematical system for rain-affected match results |
| **DRS** | Decision Review System — technology-based umpire review |
| **Playing XI** | The 11 players a team fields in a match |
| **Substitute** | A replacement fielder (cannot bat or bowl) |
| **Concussion substitute** | Full replacement (can bat and bowl) allowed for head injuries |

## Batting Terms

| Term | Definition |
|------|-----------|
| **Batting average** | Total runs / number of dismissals |
| **Strike rate (SR)** | (Runs scored / Balls faced) × 100 |
| **Boundary** | Ball reaching or crossing the rope (not in the air): 4 runs |
| **Six** | Ball clearing the rope in the air: 6 runs |
| **Duck** | Dismissed for 0 runs |
| **Golden duck** | Out first ball without scoring |
| **Half-century (50)** | Scoring 50 or more runs in an innings |
| **Century (100)** | Scoring 100 or more runs in an innings |
| **Batting order** | Sequence in which a team's batters come in to bat (1 = opener, 11 = last) |
| **Opener** | Batter who starts the innings at position 1 or 2 |
| **Anchor** | Batter who bats through the innings with consistency (low SR, high average) |
| **Finisher** | Batter who specialises in batting at the end of an innings (positions 6–8) |
| **Left-hand bat (LHB)** | Batter who bats with left hand dominant (plays from right side of crease) |
| **Right-hand bat (RHB)** | Batter who bats with right hand dominant |
| **Pinch-hitter** | Lower-order batter promoted to hit aggressively early in an innings |

## Bowling Terms

| Term | Definition |
|------|-----------|
| **Economy rate** | Runs conceded per over bowled |
| **Bowling average** | Runs conceded / wickets taken |
| **Bowling strike rate** | Balls bowled / wickets taken (lower = better) |
| **Maiden over** | Over in which no runs are scored |
| **Dot ball** | Legal delivery from which no runs are scored |
| **Wicket** | Dismissal of a batsman |
| **LBW** | Leg Before Wicket — ball would have hit the stumps but struck the batsman's body first |
| **Caught** | Batsman caught by a fielder before the ball bounces |
| **Bowled** | Ball hits the stumps directly from the bowler |
| **Run out** | Batsman dismissed when not in their crease while the ball is live |
| **Stumped** | Wicketkeeper dismisses batsman out of crease while not playing a shot |
| **Hit wicket** | Batsman dislodges the stumps with bat or body |
| **Pace bowler** | Bowler who generates speed as primary weapon (>120 km/h fast, >110 medium-fast, >90 medium) |
| **Spin bowler** | Bowler who generates turn (rotation) as primary weapon |
| **Off-spin (OB)** | Right-arm finger spin turning from off to leg for right-handed batters |
| **Left-arm orthodox (SLA)** | Left-arm finger spin; mirror of off-spin |
| **Leg-spin (LB/LBG)** | Wrist spin turning from leg to off; googly is the variation |
| **Chinaman** | Left-arm wrist spin |
| **Swing bowling** | Ball moving in air due to aerodynamic forces |
| **Seam bowling** | Ball deviating off the pitch seam on landing |
| **Death bowling** | Specialty of bowling effectively in the final overs |
| **Powerplay bowling** | Bowling in fielding restriction overs (overs 1–6 in T20) |

## Fielding Terms

| Term | Definition |
|------|-----------|
| **Wicketkeeper (WK)** | Specialist fielder behind the stumps; can take catches and stumpings |
| **Catch** | Fielder takes a clean catch before ball bounces; batter dismissed |
| **Stumping** | Wicketkeeper knocks bails off when batsman is outside crease |
| **Run out (direct)** | Fielder throws and hits the stumps directly without being relayed |
| **Run out (indirect)** | Ball relayed through another fielder or wicketkeeper before hitting stumps |
| **Substitute fielder** | Player fielding in place of an injured player; cannot bat or bowl |

## Fantasy Cricket Terms

| Term | Definition |
|------|-----------|
| **Dream11** | India's largest fantasy cricket platform; XithSense optimises for Dream11 scoring |
| **Fantasy points (FP)** | Score awarded based on real player performance per Dream11 rules |
| **Captain (C)** | Fantasy team captain; earns 2× FP for the match |
| **Vice-Captain (VC)** | Fantasy team vice-captain; earns 1.5× FP |
| **Credits** | In-game currency used to select players; each team has 100 credits |
| **Ownership** | Fraction of contest entries that include a given player |
| **Differential** | Low-ownership pick (< 20%) with high upside; used in grand leagues |
| **Lock pick** | Player so strong they are included in virtually all teams (> 75% ownership) |
| **Grand League (GL)** | Large contest (thousands of entries); high prize, low per-entry win probability |
| **Small League / Head-to-head** | Small contest with few entries; higher win probability |
| **Fantasy ceiling** | A player's best realistic FP score (90th percentile of history) |
| **Fantasy floor** | A player's worst realistic FP score (10th percentile of history) |
| **FP consistency** | Coefficient of variation of FP; lower = more consistent performer |
| **Safe team** | Conservative team optimising for expected FP; minimises variance |
| **Aggressive team** | Team maximising upside (ceiling FP); accepts higher variance |
| **Punt captain** | High-risk, low-ownership captain pick for maximum upside in GL |
| **Contrarian captain** | Captain choice opposite to popular consensus |
| **Split captain** | Distributing different captain choices across multiple GL entries |
| **ROI** | Return on investment; total winnings minus total entry fees |
| **Double-ownership** | Selecting a player as both captain in some teams and VC in others |

## XithSense Internal Terms

| Term | Definition |
|------|-----------|
| **Ensemble score** | Weighted combination of ML + human rules + form + live context |
| **Human rules engine** | Rule-based system encoding analyst knowledge as conditional adjustments |
| **Rule trigger** | Instance of a human rule activating for a player in a specific context |
| **Rolling feature** | Statistical feature computed over N most recent matches (window = 3, 5, or 10) |
| **Feature vector** | 47-dimensional array representing a player's statistical profile |
| **LP optimizer** | Linear programming team optimizer (PuLP library) |
| **DEAP fallback** | Genetic algorithm optimizer activated when LP is infeasible |
| **Match phase** | Stage of prediction: `pre_toss`, `post_toss`, `live` |
| **Cricsheet key** | Unique player identifier in Cricsheet data e.g. `"Kohli, V"` |
| **Bowler type** | Classification of bowling style: `pace_right`, `pace_left`, `spin_off`, `spin_left`, `spin_wrist` |
| **Venue alias** | Alternative name for the same venue used in different seasons |
| **Feature drift** | Distribution shift in input features over time; detected by PSI |
| **Prediction drift** | Distribution shift in model output scores over time |
| **Model bundle** | Set of 3 trained models (XGBoost + LightGBM + CatBoost) for a format |
| **Hot reload** | Replacing an active model in memory without restarting the API |
| **Backtest** | Replay of predictions against historical matches with known outcomes |
| **Captain accuracy (CA)** | Fraction of matches where predicted best captain = actual top scorer |
| **Correct player rate (CPR)** | Fraction of matches where ≥ 7 of 11 predicted players match actual top-11 |

## Statistical Terms

| Term | Definition |
|------|-----------|
| **MAE** | Mean Absolute Error — average absolute difference between predicted and actual values |
| **RMSE** | Root Mean Squared Error — square root of average squared differences |
| **Spearman ρ** | Rank correlation coefficient (−1 to +1); measures how well predicted rank matches actual rank |
| **CV** | Coefficient of Variation = standard deviation / mean; measures relative variability |
| **Percentile** | Value below which a given percentage of observations fall |
| **PSI** | Population Stability Index — measures distribution shift between two samples |
| **Z-score** | Number of standard deviations from the mean; used for drift detection |
| **MinMax normalisation** | Scales values to [0, 1] range: (x − min) / (max − min) |
| **Time-series split** | Cross-validation that preserves temporal order; training always precedes validation |
| **Overfitting** | Model performs well on training data but poorly on unseen data |
| **Gradient boosting** | Ensemble ML technique that builds trees sequentially, each correcting prior errors |
