# Success Metrics

## Prediction Quality KPIs

| Metric | Definition | MVP Target | Year-1 Target |
|--------|-----------|-----------|--------------|
| Captain Accuracy | % of matches where predicted captain = actual top scorer | ≥ 38% | ≥ 42% |
| Correct Player Rate | % of matches with ≥ 7/11 correct predicted players | ≥ 58% | ≥ 65% |
| Fantasy Points MAE | Mean absolute error on predicted fantasy points | < 14 pts | < 11 pts |
| Rank Correlation (Spearman ρ) | Correlation between predicted and actual player rank | > 0.60 | > 0.68 |
