# Product Roadmap

## MVP — Q4 2026

**Theme:** Core prediction loop, IPL and T20 focus, API-first

- [x] Cricsheet data ingestion (22,062 matches)
- [x] Feature engineering pipeline
- [x] XGBoost + LightGBM + CatBoost ensemble
- [x] Human rules engine (50+ rules)
- [x] Team optimizer (4 modes)
- [x] Captain engine
- [x] LLM explainability
- [x] FastAPI backend
- [x] JWT auth, rate limiting
- [x] Telegram notifications
- [x] Razorpay subscription billing
- [x] Backtesting harness (10k matches)

---

## Phase 2 — Q1 2027

**Theme:** Real-time intelligence, React frontend, multi-format

- [ ] React SPA frontend (Match List, Team Builder, Player Profile, AI Chat)
- [ ] Live match WebSocket feed with win probability
- [ ] ODI and Test match model training
- [ ] Push notifications (Firebase)
- [ ] WhatsApp notifications
- [ ] Google OAuth
- [ ] Ownership estimation (crowd-sourced or inferred)
- [ ] Multi-language support: Hindi

---

## Phase 3 — Q2 2027

**Theme:** Advanced intelligence, data partnerships

- [ ] Live playing XI scraper (ESPNcricinfo / official feeds)
- [ ] Injury feed integration
- [ ] Odds movement tracker (Betfair/Betway API)
- [ ] Weather integration (real-time pitch impact)
- [ ] Contest ROI analytics per user
- [ ] Player travel fatigue model
- [ ] Export team to PNG/PDF for sharing

---

## Phase 4 — Q3 2027

**Theme:** Scale, mobile, marketplace

- [ ] Native iOS app (SwiftUI)
- [ ] Native Android app (Kotlin)
- [ ] Tipster marketplace (experts sell picks through XithSense)
- [ ] Multi-platform support: MyTeam11, Dream11 alternative formats
- [ ] API public tier (developers build on XithSense)
- [ ] Women's cricket prediction models (WPL, T20 World Cup)

---

## Long-Term Vision (2028+)

- AI-driven contest strategy (which contest type given your team)
- Auction draft intelligence for longer-format leagues
- International expansion: Pakistan (PSL), Australia (BBL), West Indies (CPL)
- Partnership with official cricket boards for exclusive data access
